# 🧠 Aplicação Web Acessível para Pessoas com TEA

## 📌 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma aplicação web educacional acessível, voltada para pessoas com Transtorno do Espectro Autista (TEA), baseada em princípios de Interação Humano-Computador (IHC) e no uso do hiperfoco como estratégia de engajamento.

A aplicação foi projetada para apoiar o processo de aprendizagem de forma visual, interativa e sensorialmente adequada, respeitando as necessidades cognitivas e comportamentais de pessoas com TEA.

---

## 🎯 Objetivo

Desenvolver uma aplicação web acessível que utilize elementos de hiperfoco e boas práticas de IHC para melhorar o engajamento e a experiência de aprendizagem de pessoas com TEA.

---

## 💡 Diferencial

O principal diferencial do projeto é o uso do **hiperfoco como elemento central de design**, explorando:

* Personagens de interesse do usuário
* Feedbacks visuais e auditivos suaves
* Redução de estímulos distratores
* Interações simples e previsíveis
* Dois modos de uso:

  * **Modo simples** (menos estímulos)
  * **Modo com personagem (hiperfoco)**

---

## 🧩 Funcionalidades

* Aprendizado de vogais (A, E — em expansão)
* Reprodução de áudios educativos
* Atividades interativas com feedback gentil
* Sistema de progressão por etapas
* Interface adaptável ao perfil do usuário
* Modo acessível (redução de estímulos visuais)

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## 🧪 Metodologia

O projeto segue uma abordagem iterativa baseada em:

1. Pesquisa sobre TEA, IHC e hiperfoco
2. Criação de protótipos (Canva)
3. Implementação da aplicação web
4. Testes com usuários reais (inicialmente adulto com TEA)
5. Coleta de feedback e ajustes de usabilidade

---

## 📊 Resultados Esperados

* Melhor engajamento durante atividades educacionais
* Redução de frustração em interações
* Maior facilidade de foco e compreensão
* Validação do hiperfoco como estratégia de design

---

## ⚠️ Limitações

* Número reduzido de participantes nos testes iniciais
* Aplicação ainda em fase de expansão (conteúdo limitado)
* Dependência de dispositivos (ex: tablet) para melhor experiência

---

## 🚀 Próximos Passos

* Expandir para mais letras e conteúdos
* Testes com mais usuários (ex: instituições como APAE)
* Melhorias na acessibilidade e personalização
* Publicação da aplicação online
