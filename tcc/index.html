Projeto de TCC

Aplicacao Web Acessivel para Pessoas com TEA com foco em IHC e hiperfoco.

Espaco reservado para organizar contexto, metas de validacao, prototipos e resultados de testes em tablet com usuarios autistas.

Resumo do Tema

Desenvolvimento de uma aplicacao web acessivel para pessoas com Transtorno do Espectro Autista, baseada em principios de Interacao Humano-Computador e no uso do hiperfoco como estrategia para aumentar engajamento educacional.

Objetivos

  • Aprimorar acessibilidade cognitiva
  • Validar interface em tablet
  • Usar feedback de usuarios reais

Roadmap Inicial

  • Prototipo de alta fidelidade
  • Teste piloto assistido
  • Coleta de metricas de engajamento
